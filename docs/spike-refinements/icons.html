BOI Site Icon Library

Inline SVG icons for section headers, quick-take clusters, overview cards, and misc UI. All stroke-based, using currentColor for CSS color control.

Usage: Copy the SVG markup inline. Add the appropriate class (.section-icon, .cluster-icon, .card-icon, .inline-icon) to control sizing. Icons inherit color from their parent element via currentColor.

Integration Reference

How to integrate into index.html:

Section headers:
<h2 class="section-title"><svg class="section-icon" ...>...</svg> The Comfortable Illusion</h2>

Quick-take clusters:
<div class="cluster-header"><svg class="cluster-icon" ...>...</svg> <span>The Big Picture</span></div>

Overview cards:
<div class="card-header"><svg class="card-icon" ...>...</svg> <span>Act 1: The Problem</span></div>

CSS (add to styles.css):
.section-icon { width: 28px; height: 28px; vertical-align: middle; margin-right: 0.5rem; color: currentColor; flex-shrink: 0; }
.cluster-icon { width: 22px; height: 22px; color: currentColor; flex-shrink: 0; }
.card-icon    { width: 24px; height: 24px; color: currentColor; flex-shrink: 0; }
.inline-icon  { width: 20px; height: 20px; vertical-align: text-bottom; color: currentColor; flex-shrink: 0; }
